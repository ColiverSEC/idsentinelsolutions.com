Overview

This module provides hands-on experience with cross-platform IAM automation. It demonstrates how automation can orchestrate identity lifecycle tasks across multiple systems including Active Directory, Microsoft Entra ID, and Okta. Key focus areas include provisioning, deprovisioning, lifecycle workflows, compliance reporting, self-service automation, and logging/error handling.
